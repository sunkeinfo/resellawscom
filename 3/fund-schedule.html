完整规则体系

🎯 最高指令：最小化闲置期优先

所有决策的出发点是让资金尽快进入下一个定期存款，最大限度减少其停留在活期账户的时间。这意味着我们放弃对特定银行的偏好，也放弃长时间的"战略性持有"。

💰 资金净增量原则

核心法则：银行判断新资金的唯一标准是您在特定计算周期内，名下所有账户的资金净增长额。

公式：合格新资金 = 周期内总转入 - 周期内总转出

  • 单纯的资金转入不算新资金
  • 必须被同周期内的资金转出所抵消
  • 日内互换无效（同天转入转出净值为零）
  • 新资金必须来源于其他银行

⚠️ 冷却期触发原则

关键规则：只有转出资金的银行才会进入冷却期，接受资金的银行状态不变。

  • 转出触发：任何资金转出立即触发冷却期
  • 转入无影响：接受资金不会影响银行状态
  • 冷却期内：无法提供"新资金"资格
  • 等待期：必须等待完整冷却期结束
  • 无法提前：不可通过任何操作提前解除

🏦 银行专属计算周期

汇丰银行 (HSBC)

冷却期：7个历日

计算方式：从转出日起算7天

示例：9月10日转出 → 9月17日恢复

恒生银行 (Hang Seng)

冷却期：15个历日

计算方式：从转出日起算15天

示例：9月10日转出 → 9月25日恢复

中国银行 (BOC)

冷却期：按自然月计算

计算方式：整个转出月份都处于冷却期

示例：9月任意日转出 → 10月1日恢复

⚡ 执行策略原则

🚀 即时行动原则

  • 资金到期时立即检查所有银行状态
  • 存在任何"干净"银行就立即转入
  • 不做任何等待或犹豫
  • 优先选择冷却期最短的银行

⏳ 最短等待原则

  • 仅在所有银行都处于冷却期时才等待
  • 精确计算最早恢复"干净"状态的银行
  • 设定延迟转账目标日期
  • 等待时间压缩到理论最小值

🎯 初始资金激活策略

问题：现有旧资金（非新资金）如何转化为新资金？

解决方案：将所有旧资金统一转入一个第三方"干净"银行

具体操作步骤：

  1. 选择一个当前"干净"的银行作为接收方（推荐汇丰）
  2. 将恒生10,000USD转入汇丰
  3. 将中银20,000USD转入汇丰
  4. 确保接收银行只有净流入，无转出
  5. 所有旧资金无可争议地转化为新资金

🔄 银行状态转换规则

✅ 干净状态 → 冷却状态

触发条件：银行转出任何金额的资金

生效时间：转出操作完成后立即生效

注意：接受资金不会改变银行状态

❄️ 冷却状态 → 干净状态

恢复条件：完整冷却期结束

时间计算：从转出日开始计算

无法提前：任何操作都无法缩短冷却期

🛡️ 规避冷却期的关键要点

✅ 正确做法

  • 始终保持至少一个银行处于"干净"状态
  • 轮换使用不同银行，避免同时污染
  • 优先使用冷却期最短的银行
  • 严格按照净增量原则操作
  • 精确计算和跟踪冷却期结束时间
  • 关键：只有转出才触发冷却期，接受资金无影响

❌ 禁止行为

  • 同一天内转入后又转出同一银行
  • 在银行内部账户间转移资金
  • 将资金转出后立即转回同一银行
  • 忽视冷却期强行转入
  • 为了偏好某银行而延长等待时间
  • 误区：认为接受资金也会触发冷却期
